# pet-care-management-system
PetCare Scheduler is a Java-based application that allows users to register pets, manage appointments, and generate reports. The project demonstrates strong Object-Oriented Design, file handling through serialization, Java Collections, and modern Date-Time API usage for real-world scheduling scenarios.

## Features
- Pet registration system
- Appointment scheduling with validation
- Future date enforcement
- Report generation
- File persistence using serialization
- Console-based interactive menu

## Tech Used
- Java
- OOP
- HashMap & ArrayList
- LocalDateTime API
- Serialization
